## 减速带绘制

1. 单击顶部 **减速带** 按钮。

2. 底图上单击第一个点确定减速带起点。

3. 单击第二个点确定减速带的终点，完成减速带绘制。
   
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_5a1615e.png)

## 移动减速带

1. 选中想要移动的减速带。
2. 鼠标拖动可进行减速带移动。

## 减速带删除

1. 选中想要删除的减速带。

2. 单击 Delete 键删除减速带。
   
   - Windows 系统按 Delete 键删除。
   - MAC OS 系统按 fn + Delete 键删除。
