## 新增路沿

1. 单击 **路沿** 按钮。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_e6c41ab.png)

2. 单击选定的车道一侧创建路沿起点，单击选定的车道另一侧创建路沿终点。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_f2224d1.png)
   
   ## 删除路沿

3. 单击选中想要删除的路沿。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_e3839ac.png)
   
   选中后路沿变为蓝色。

4. 点击 **Delete** 键删除路沿。
   
   - Windows 系统按 **Delete** 键删除。
   - MAC OS 系统按 **fn** + **Delete** 键删除。
