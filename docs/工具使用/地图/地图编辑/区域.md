## 新增区域

### 新增四边形区域

1. 单击 **区域** 按钮。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_0e3cdd6.png)

2. 点击选定的区域，顺序逐个绘制点，相邻点之间连线，双击/Esc键/Enter键结束绘制，尾点与首点自动相连。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_4cf5ee6.png)
   
   > 说明：请至少绘制三个点。
   
   ### 新增多边形区域

3. 单击 **区域** 按钮。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_0e3cdd6.png)

4. 点击选定的区域，顺序逐个绘制点，相邻点之间连线，双击/Esc键/Enter键结束绘制，尾点与首点自动相连。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_7234ce1.png)
   
   > 说明：请至少绘制三个点。

## 更改区域属性

1. 单击选中您想要修改的车道属性。

2. 在右侧 **属性** 中编辑您想要修改的区域属性。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_c5bfb78.png)
   
   ## 删除区域

3. 单击选中想要删除的区域。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_03c1e24.png)
   
   选中后区域会变成蓝色。

4. 点击 **Delete** 键删除区域。
   
   - Windows 系统按 **Delete** 键删除。
   - MAC OS 系统按 **fn** + **Delete** 键删除。
