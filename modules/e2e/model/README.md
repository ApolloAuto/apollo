# Quick Start Guide
## Train Model

The train model

	sh run_train.sh
Download the trainsets to the current project.  See `./trainsets` directory.

## Model predict

The predict model

	sh run_predict.sh path/to/steering_model path/to/acc_model path/to/test_img t/path/to/result
