## 新增道闸

1. 单击 **道闸** 按钮。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_2210143.png)

2. 单击选定的车道一侧创建停止线起点，单击选定的车道另一侧创建停止线终点，创建停止线。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_898cd7a.png)

3. 在车道方向距停止线一定距离处（默认长度2.5m）创建道闸，道闸位置对齐停止线中点。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_96f5b1e.png)
   
   ## 移动道闸

4. 单击选中道闸。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_a3cf089.png)
   
   选中后道闸变成蓝色。

5. 移动停止线和道闸至指定位置。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_503851a.png)

## 查看道闸

1. 单击选中道闸。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_a3cf089.png)

2. 查看右侧属性信息。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_d6e5e30.png)
   
   ## 更改道闸属性

3. 单击选中道闸。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_a3cf089.png)

4. 在右侧修改属性信息。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_d6e5e30.png)
   
   ## 删除道闸

5. 单击选中道闸。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_a3cf089.png)

6. 点击 **Delete** 键删除道闸，同步删除关联的停止线。
   ![image.png](https://bce.bdstatic.com/doc/Apollo-Homepage-Document/Apollo_Beta_Doc/image_05812bf.png)
   
   - Windows 系统按 **Delete** 键删除。
   - MAC OS 系统按 **fn** + **Delete** 键删除。
